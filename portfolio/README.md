# Personal Portfolio Website

A modern, minimalist, and responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Smooth Scrolling Navigation**: Single-page design with smooth scroll to sections
- **Active Section Highlighting**: Navigation automatically highlights the current section while scrolling
- **Hamburger Menu**: Mobile-friendly collapsible navigation menu
- **Fade-in Animations**: Intersection Observer API for smooth element animations on scroll
- **Accessible**: Semantic HTML and keyboard navigation support
- **Performance Optimized**: Lazy loading images and efficient CSS

## Color Scheme

- **Noir** (#000000) - Main background
- **Midnight** (#122C4F) - Primary accent
- **Ocean** (#5B88B2) - Secondary accent and active states
- **Pearl Perfect** (#FBF9E4) - Text and light sections

## Sections

1. **About Me** - Introduction and overview
2. **Education** - Academic background and certifications
3. **Hobbies** - Personal interests and activities

5. **IT Experience** - Technical skills and expertise

## How to Use

1. Open `index.html` in any modern web browser
2. No build process or server required
3. All assets are linked via CDN or inline
